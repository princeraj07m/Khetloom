Usage Instructions
How to use the FieldLayoutComponent:
  1. Import the component in your module
  2. Use the component in your template: <app-field-layout [plants]="yourPlantArray"></app-field-layout>
  3. Pass a Plant array with the required structure
Plant Interface:
interface Plant {{ '{' }}
  plantId: string;
  x: number;
  y: number;
  status?: 'healthy' | 'defected';
{{ '}' }}
Example JSON Input:
[
  {{ '{' }} "plantId": "P-1-1", "x": 1, "y": 1, "status": "healthy" {{ '}' }},
  {{ '{' }} "plantId": "P-1-2", "x": 2, "y": 1, "status": "defected" {{ '}' }},
  {{ '{' }} "plantId": "P-2-1", "x": 1, "y": 2, "status": "healthy" {{ '}' }}
]
