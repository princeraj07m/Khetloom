BASE\_URL = [https://khetloom-backend.vercel.app](https://khetloom-backend.vercel.app)

---

**Health:**

* GET [https://khetloom-backend.vercel.app/api/health](https://khetloom-backend.vercel.app/api/health) → ok, env, time

**Auth:**

* POST [https://khetloom-backend.vercel.app/api/register](https://khetloom-backend.vercel.app/api/register) → creates user; returns token and user
* POST [https://khetloom-backend.vercel.app/api/login](https://khetloom-backend.vercel.app/api/login) → returns token and user

**Profile (requires Authorization: Bearer <token>):**

* GET [https://khetloom-backend.vercel.app/api/profile](https://khetloom-backend.vercel.app/api/profile) → current user
* PUT [https://khetloom-backend.vercel.app/api/profile](https://khetloom-backend.vercel.app/api/profile) → update current user

**Users (requires token):**

* GET [https://khetloom-backend.vercel.app/api/users](https://khetloom-backend.vercel.app/api/users) → list users
* GET [https://khetloom-backend.vercel.app/api/users/\:id](https://khetloom-backend.vercel.app/api/users/:id) → single user by id

**Public dashboards (no auth):**

* GET [https://khetloom-backend.vercel.app/api/public/stats](https://khetloom-backend.vercel.app/api/public/stats) → { totalUsers, avgFarmSize, avgMonthlyExpenditure }
* GET [https://khetloom-backend.vercel.app/api/public/recent-users?limit=10](https://khetloom-backend.vercel.app/api/public/recent-users?limit=10) → latest users (limited fields)
* GET [https://khetloom-backend.vercel.app/api/summary](https://khetloom-backend.vercel.app/api/summary) → { totalUsers, newThisWeek, avgFarmSize }
* GET [https://khetloom-backend.vercel.app/api/app-data](https://khetloom-backend.vercel.app/api/app-data) → full analytics + users
* GET [https://khetloom-backend.vercel.app/api/public/config](https://khetloom-backend.vercel.app/api/public/config) → { environment, version, time }
